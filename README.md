## window10风格的管理系统

- **前端** 主要部分引用element-ui,vue,vuex构建
-  **后端** KOA2


#### 构建和设置

``` bash
# install dependencies
npm install

# 启动前端界面
npm run dev

# 启动服务端
npm run server

# 同时启动前后端
npm run start
# build for production with minification
npm run build

```
